# creo-work — личная рабочая папка Микиты

Синхронизируется между компами через git (приватный репо). Не канон — личные рабочие артефакты.

- `shotlists/` — микродрама-шотлисты (.md + .html)
- дальше сюда же — любые личные рабочие файлы

## Синк
- `worksync` — подтянуть свежее + закоммитить/запушить твои изменения (одна команда).
- Новый комп (разово): `git clone <repo-url> ~/creo-work`, затем всегда `worksync`.
